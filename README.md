# webpack-vue-multi-page

> A Vue.js project

## 安装依赖

* npm install

## 本地启动

* npm run dev

## 生成部署文件

* npm run build

## 目录结构

* config：webpack 配置文件
* node_modules：第三方依赖文件
* src：源码
* static:公共文件（vue.js vuex.js等）
* wwwroot：打包输出文件

## 源码结构

* assets：静态资源文件
* components：vue 组件，.vue 文件可以包含 html，js 和 css，打包时会分开输出
* pages：多页面
